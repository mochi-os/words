# Contributing to Mochi

Mochi is free and open source software, developed by invitation by a small team. The source is always available and anyone is free to fork it, but code is written only by Mochi OÜ and a small number of invited contributors - for a project of this size that is faster, more reliable, and keeps the copyright in Mochi clean.

Feature requests and bug reports, described in words, are welcome. Good ones are implemented quickly. Please do not send pull requests or post code, patches, or snippets. To keep the copyright clean we cannot use them. Unsolicited contributions are closed or left unread.

Contact: **developers@mochi-os.org**
